# Melodist

## Пояснительная записка

Автор: Оганян Тигран, Анна Гордецкая, Даниил Слюсарь



### Описание проекта:
Melodist - веб приложение, в котором вы можете создавать плейлисты со своей любимой музыкой.
#### В чем задумка: 
- составлять собственные подборки, искать тот или иной трек, для дальнейшего прослушивания на музыкальной площадке
- при создании профиля, вы должны указать направления в музыке, а также исполнителей на ваш вкус
- ИИ будет составлять подборки из музыкальных композиций, основанных на ваших вкусах(музыкальный "тиндер")
- лайкайте и добавляйте в плейлисты музыку, что попалась в подборке
- с помощью поиска находите треки, и добавляйте их в свои плейлисты
- также в поиске находите уже существующие пользователей и их плейлисты
