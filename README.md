### Описание проекта:
Melodist - веб приложение, в котором вы узнаете новые грани в музыке

#### В чем задумка: 
- при создании профиля, вы должны указать направления в музыке, а также исполнителей на ваш вкус
- ИИ будет составлять подборки из музыкальных композиций, основанных на ваших вкусах(музыкальный "тиндер")
- лайкайте и добавляйте в плейлисты музыку, что попалась в подборке
- с помощью поиска находите треки, и добавляйте их в свои плейлисты
- также в поиске находите уже существующие пользователей и их плейлисты

#### Как запустить приложение, чтобы оно работало
для работы приложения, запустите файл app.py, а также убедитесь что у вас есть работающая ссылка для подклюения к LLM
если у вас такой нет, то попросите ссылку и вставьте ее в код. после передите по 2 ссылке, для того чтобы код рабатал по нелокальной ссылке


## Техническое задание
- создать класс пользователя
  - метод регистрации, входа и выхода из профиля
  - подключение к бд
  - создание страницы для регистрации, входа в аккаунт
  - создание странички для заполнения музыкальных данных после регистрации(обязательна для заполнения)
  - метод создания плейлистов, возможность создания плейлистов для себя(скрытый), или общедоступный(открытый)
- создание страницы музыкальной подборки (музыкального тиндера)
  - работа с ии(запросы в локальную нейросеть) + бд и музыки
  - создание метода лайка, дизлайка, добавление в плейлист
- поисковая строка
  - метод поиска музыкальных композиций, пользователей, пользовательские открытые плейлисты
  - возможность добавлять других пользователей в друзья 
- создание профиля пользователя
  - метод отображения друзей, плейлистов
